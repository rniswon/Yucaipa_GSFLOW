•	The model dataset is not final as it might be changed as part of the review process. 
•	The model simulates the period 1947-2014. The period 1947-1989 is a spin-up period.
•	The model is calibrated for the period 1990-2014.
•	 The model relays on new features in GSFLOW 2.0 that are still not released; therefore, the model might subject to changes as part of the release. 
•	The model can be run using the batch file “gsflow_yuc.bat”.
•	Some of the output files such as the cell-by-cell budget file was removed from the model folder because of its huge size; however, a complete set of model output can be generated by running the model on your local computer.
